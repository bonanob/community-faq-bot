# Community FAQ Bot 🏠

A bilingual (English/German) AI-powered FAQ chatbot for community room bookings.
Ask questions naturally in either language and get accurate answers instantly.

---

## The Problem

Managing a community room means answering the same questions repeatedly:
- What does the room cost on weekends?
- How many people does it fit?
- What's the cancellation policy?
- Is the room accessible?

These questions come in by email, phone, and in person — taking up time that
could be spent on more meaningful community work. This bot answers them
instantly, in the user's language, 24/7.

---

## How It Works

This project uses **RAG (Retrieval-Augmented Generation)** combined with
**AI agents** to answer questions accurately based on a custom knowledge base.

    User question (EN or DE)
            ↓
    Language Detector Agent
            ↓
    Retriever Agent → searches ChromaDB knowledge base
            ↓
    Answer Agent → generates response from retrieved context
            ↓
    Reviewer Agent → checks accuracy and groundedness
            ↓
    Answer returned in user's language

The bot only answers from the knowledge base — it never makes things up.
If a question isn't covered, it says so honestly and suggests contacting
the community directly.

## Tech Stack

| Component | Tool | Why |
|---|---|---|
| Agent framework | CrewAI | Beginner-friendly, clean agent orchestration |
| AI model | MiniMax M2.5 via OpenRouter | Free tier for development, cheap in production |
| Vector database | ChromaDB | Local, free, simple Python integration |
| Embeddings | OpenAI text-embedding-3-small | Multilingual, cheap, reliable |
| UI | Streamlit | Pure Python, fast to build |
| Language | Python 3.11.9 | Stable, broad library support |

## Getting Started

### Prerequisites
- macOS with Homebrew installed
- Python 3.11.9 via pyenv
- OpenRouter account (free tier works)

### Installation

```bash
# Clone the repo
git clone git@github.com:bonanob/community-faq-bot.git
cd community-faq-bot

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env and add your OpenRouter API key
```

### Run the app
```bash
streamlit run src/app.py
```
